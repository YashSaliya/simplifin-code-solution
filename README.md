# Ride-Sharing Application

This is a simple ride-sharing application implemented in Python. It allows users to offer shared rides, select available rides, and manage ride statistics.

## Features

### User Management
Users can register and manage their vehicles.

### Vehicle Management
Users can add details about their vehicles, including category, number, and total seats.

### Ride Offering
Users can offer shared rides on specific routes with available seats.

### Ride Selection
Users can search and select from multiple available rides based on their preferences.

### Ride Management
Users can end a ride once it's completed.

### Statistics
The application provides functionality to print ride statistics for all users.
